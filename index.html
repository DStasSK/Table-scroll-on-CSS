Фиксированный заголовок и первая
колонка таблицы - на чистом CSS.

Ниже представлена стилизация таблицы у которой:
  • первая строка (или группа строк) и первый столбец будут зафиксированы относительно элемента документа;
  • основное содержимое будет свободно перемещаться внутри таблицы;
  • зафиксированные элементы будут согласованно смещаться относительно содержимого таблицы;
  • JavaScript не будет использован - только CSS;
  • сама таблица должна находиться в ограничивающей обертке.

Итак, приступим. Есть большая таблица, которую нужно поместить в маленьком окошке. При просмотре таблицы, нам необходимо видеть первую ячейку в строке и первую ячейку в столбце для любой ячейки. На данном этапе мы имеем базовую таблицу с минимальным оформлением.

base table
<div class="tableBox">
  <table class="table"><tbody>
    <tr><th>~</th><th>1</th><th>2</th>...<th>20</th></tr>
    <tr><td>A</td><td>A1</td><td>A2</td>...<td>A20</td></tr>
    ...
  </tbody></table>
</div>
			
<style>
  .tableBox { overflow:auto; height:250px; }
  .table { border-collapse: collapse }
</style>

Стилизованная таблица

scroll table
<div class="tableBox">
  <table class="scroll_table"><tbody>
    ...
  </tbody></table>
</div>

<style>
  .scroll_table {
    border-collapse: collapse;
    border:solid transparent; border-width:0 .1px .1px 0;   /* (1) */
    --th_bg-color: #b0e0e6;   /* th background-color */
    --th1_bg-color: #5F9EA0;  /* th(1) background-color */
    --td_bg-color: #fffacd;   /* td background-color */
    --td1_bg-color: #f0e68c;  /* td(1) background-color */
    --b-size: 1px;            /* border-size через размер box-shadow */
    --b-size-n: -1px;         /* border-size */
    --b-color: #000;          /* border-color */
  }

  .scroll_table th, .scroll_table td { border: none }
  .scroll_table th { background: var(--th_bg-color) }
  .scroll_table td { background: var(--td_bg-color) }
  .scroll_table tr th:nth-child(1) { background: var(--th1_bg-color) }
  .scroll_table tr td:nth-child(1) { background: var(--td1_bg-color) }

  .scroll_table th {
    position:sticky; top:0; z-index:3;
    box-shadow: inset var(--b-size-n) var(--b-size-n) 0 var(--b-color),
                inset 0 var(--b-size) 0 var(--b-color);
  }
  .scroll_table tr th:nth-child(1) {
    left:0; z-index:4;
    box-shadow: inset var(--b-size-n) var(--b-size-n) 0 var(--b-color),
                inset var(--b-size) var(--b-size) 0 var(--b-color);
  }
  .scroll_table td {
    box-shadow: inset var(--b-size-n) var(--b-size-n) 0 var(--b-color)
  }
  .scroll_table tr td:nth-child(1) {
    position:sticky; left:0; z-index:2;
    box-shadow: inset var(--b-size-n) var(--b-size-n) 0 var(--b-color),
                inset var(--b-size) 0 0 var(--b-color);
  }
</style>
(1) - добавил правую и нижнюю прозрачную границу таблицы так как иногда пропадали соответствующие границы у последних ячеек. Думаю что проблема на аппаратном уровне, или же все дело в магии, а именно - в магии иллюзии ;-)

Для решения поставленной задачи использовалось свойство position со значением sticky. При таком значении, элемент рассматривается как позиционированный относительно, пока он не пересекает определённый порог, после чего рассматривается как фиксированный. position:sticky работает только, если родительский блок имеет назначенную высоту height или min-height.

Значение свойства z-index для ячеек в первой строке должно быть выше, чем в последующих строках. Для них устанавливаем z-index:3. У первой ячейки в первой строке значение должно быть еще больше - z-index:4.

Для имитации свойства border мы будем использовать свойство box-shadow. Если мы будем использовать комбинацию из двух этих свойств, то у нас будет смещение создаваемых границ на стыках ячеек.



Так же при задании границ ячеек через border:
  • границы ячеек в первой строке и в первом столбце пропадают;
  • наша таблица будет сдвигаться вверх и влево примерно на половину от ширины границ ячеек;

Границы ячееек устанавливались с конца таблицы. Для элемента <td> - нижнюю и правую (рис. a). В первой колонке для элемента <td> дополнительно устанавливается левая граница (рис. b). В заголовках столбцов (элемент <th>) - дополнительно устанавливается верхняя граница (рис. c). В первой ячейке таблицы - все четыре границы. (рис. d).

a .tt-4 td {
  box-shadow:inset -1px -1px 0 #000;
}
b .tt-4 tr td:nth-child(1) {
  box-shadow:inset -1px -1px 0 #000,
             inset 1px 0 0 #000;
}
c .tt-4 th {
  box-shadow:inset -1px -1px 0 #000,
             inset 0 1px 0 #000;
}
d .tt-4 tr th:nth-child(1) {
  box-shadow:inset -1px -1px 0 #000,
             inset 1px 1px 0 #000;
}
Добавлю пару моментов.
  • Если мы установим более толстые рамки, то необходимо добавить соответствующие padding, иначе содержимое будет отображаться со смещением. Разницу в 1px заметить сложно, но она есть.
  • Вместо свойства box-shadow мы можем использовать псевдоэлемент ::before у элементов таблицы и устанавливать свойство border для ::before. Так легче регулировать ширину границ ячеек, но мы получим больше кода и некоторые артефакты :-)

Для более сложной структуры заголовка таблицы, все элементы таблицы <th> объединим при помощи элемента <thead>, все остальные ячейки <td> объединим при помощи элемента <tbody>. Данная структура поможет нам проще управлять сложным заголовком таблицы.

scroll table on thead
<div class="tableBox">
  <table class="scroll_table">
    <thead><!-- th zone --></thead>
    <tbody><!-- td zone --></tbody>
  </table>
</div>

<style>
  .tableBox { overflow:auto; height:60vh; }
  .scroll_table {
    border-collapse:collapse;
    border:solid transparent; border-width:0 .1px .1px 0
  }
  .scroll_table thead{ position:sticky; top:0; z-index:3 }

  .scroll_table th, .scroll_table td {
    box-shadow:inset -1px -1px 0 #000
  }
  .scroll_table tr:nth-child(1) th {
    box-shadow:inset -1px -1px 0 #000, inset 0 1px 0 #000
  }
  .scroll_table tr:nth-child(1) th:nth-child(1) {
    position:sticky; left:0; z-index:4;
    box-shadow:inset -1px -1px 0 #000, inset 1px 1px 0 #000
  }
  .scroll_table tr td:nth-child(1) {
    position:sticky; left:0; z-index:2;
    box-shadow:inset -1px -1px 0 #000, inset 1px 0 0 #000
  }
</style>

Эту же таблицу можно сделать через задания своей предельной позиции для каждой строки с <th>.

При задании значения свойства top для элемента <th> мы должны помнить, что размеры ячеек таблицы могут измениться его содержимым.

scroll table
<div class="tableBox">
  <table class="scroll_table">
    ...
  </table>
</div>

<style>
  .tableBox { overflow:auto; height:60vh; }
  .scroll_table {
    border-collapse: collapse;
    border:solid transparent; border-width:0 .5px .5px 0
  }
  .scroll_table th { position:sticky; top:0; z-index:3; }
  .scroll_table tr:nth-child(2) th { top: 60px; }   /* уточнить */

  .scroll_table th, .scroll_table td {
    box-shadow: inset -1px -1px 0 #000;
  }
  .scroll_table tr:nth-child(1) th {
    box-shadow:inset -1px -1px 0 #000, inset 0 1px 0 #000;
  }
  .scroll_table tr:nth-child(1) th:nth-child(1) {
    position:sticky; left:0; z-index:4;
    box-shadow:inset -1px -1px 0 #000, inset 1px 1px 0 #000
  }
  .scroll_table tr td:nth-child(1) {
    position:sticky; left:0; z-index:2;
    box-shadow:inset -1px -1px 0 #000, inset 1px 0 0 #000
  }
</style>
